## ✨ المميزات الرئيسية

<div align="center">

| 🏠 الصفحة الرئيسية | 🍕 قائمة الطعام | 🛒 سلة التسوق |
|-------------------|----------------|--------------|
| قسم Hero مع خلفية بيتزا | 12 صنف بيتزا مميز | إضافة/إزالة العناصر |
| أزرار طلب سريع | تصنيفات (كلاسيك، نباتي، دجاج) | تحديث الكمية |
| تأثيرات حركية | تقييمات النجوم | حساب المجموع مع رسوم التوصيل |

| 💳 إتمام الطلب | 🌙 الوضع المظلم | 👑 لوحة التحكم |
|---------------|----------------|----------------|
| نموذج متعدد الخطوات | تبديل فوري بين الأوضاع | عرض جميع الطلبات |
| حفظ الطلبات في LocalStorage | حفظ تفضيل المستخدم | تغيير حالة الطلب |
| عرض رقم الطلب | تأثيرات انتقال سلسة | حذف الطلبات |

## 🎨 واجهات المستخدم

### 💻 سطح المكتب | 📱 الجوال | 📟 التابلت
